# QA Automation using Playwright

## Project Overview
This project demonstrates a QA Automation framework built using **Playwright (JavaScript)** for testing web applications.

## Scope of Testing
- Manual Testing
- Functional Testing
- Sanity & Regression Testing
- UI Validation
- Basic Automation Testing

## Tools & Technologies
- Playwright
- JavaScript (ES6)
- Node.js
- Git

## Test Coverage
- Login functionality (positive & negative scenarios)
- Homepage UI validation
- Form and checkbox behavior

## How to Run Tests
1. Install dependencies:
   npm install
2. Run tests:
   npx playwright test
3. View report:
   npx playwright show-report
